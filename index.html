Follow these 3 simple steps to add your own HTML5 playable.

1
Create folder
Create a new folder inside /playables/ (e.g. my-new-game)
2
Add your game
Put your game file as playable.html inside the new folder
3
Register it here
Add a new object to the playables array in the script below
